Web scraping
Predicting condominium price

This project uses Selenium library to firstly obtain all condominiums listed on the hipflat.com website, and extracts information for each page using BeautifulSoup package. This project is focused on condominium listings in Bangkok, both new and resale. This study is very useful for buyers, resellers, agents and even developers to justify the 'fair price' as a starting point based on the current actual market data.

  • github code

Sentiment Analysis
IMDB Movie Reviews

Shiny app for sentiment classification (Positive / Negative) based on IMDB movie review dataset. Classification performs with an AUC of 91%. Apart from the classification task the shiny app also provides visualizations that generate insights into the test reviews data.
